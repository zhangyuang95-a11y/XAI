# 项目目标

项目名称：面向 agent 推理系统的 explanation 形式语义框架研究

本项目的核心目标是：
对 explanation 给出严格的数学定义，并在此基础上研究 explanation 的最小性、语义充分性、解释质量以及追问链稳定性。

这是一个长期理论研究项目，不是代码工程项目。

研究重点包括：

1. explanation 的数学本体
2. 证据集合与问题之间的语义关系
3. minimal explanation
4. explanation quality
5. why-question follow-up chain
6. closure depth
7. 未来扩展方向，如 Bayesian explanation、causal explanation、SMT verification、interactive explanation agents
